# Finance-Dashboard-Power-BI-Project


## **1. Project Overview**

This project presents an interactive Finance Dashboard built in Power BI to track and analyze personal financial data, including income, expenses, and savings across multiple years (2021–2024).
The dashboard helps users understand spending behavior, monitor financial growth, and evaluate savings efficiency through visually rich insights.

## **2. Key Objectives**

Track yearly income, expenses, and savings.

Monitor financial performance over time using trend charts.

Analyze spending distribution across various categories.

Evaluate savings percentage and identify improvement areas.

Provide quick year-wise comparison for better decision-making.

## **3. Dashboard Features**
   
3.1 Income Trend Analysis

A line chart displaying income progression from 2021 to 2024.

Highlights major percentage changes (e.g., 35%, 53%, 48%, 58%).

Tabs to switch between Income, Expenses, Savings, and Target views.

Helps identify seasonal patterns and financial growth.

3.2 Summary Cards

₹ 3M – Total Income

₹ 1M – Total Expenses

₹ 2M – Total Savings

64% Savings Rate
These KPI cards provide a quick snapshot of overall financial health.

3.3 Year-wise Filters

Interactive buttons for:

All, 2021, 2022, 2023, 2024
Allows dynamic switching between years for focused analysis.
